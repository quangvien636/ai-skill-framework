# Draft to Reviewed Package

This production Workflow maps the Review Quality v1 sequence:

Draft → structure → clarity → evidence alignment → tone → platform fit → CTA →
required revisions → review report → approve, revise, or reject.

The detailed checks remain in the Skill procedure because the Workflow schema
supports Skill invocation steps rather than free-standing reasoning nodes. This
Workflow performs no model execution, browsing, external fact-checking, or
publishing.
