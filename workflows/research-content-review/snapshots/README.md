# Golden Snapshots

These files are deterministic compiler proofs:

- `composite-workflow.json` fixes the three-Skill topology and artifact flow.
- `composite-binding.json` fixes step-to-Runtime resolution.
- `composite-execution-plan.json` fixes plan order, batches, and outputs.
- `compiled-state-graph.json` fixes compile-only graph topology and metadata.
- `reviewed-content-package.json` fixes the canonical editorial artifact shape.

The reviewed package is an explicitly static fixture with a `revise`
recommendation. It demonstrates shape without claiming that Research, Content
Creation, Review Quality, rendering, or publishing was executed.
