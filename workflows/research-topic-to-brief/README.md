# Research Topic to Brief

This Workflow maps the full Research v1 method into one production Skill step:

Topic → clarify questions → define source requirements → organize supplied
evidence → rank source reliability → map claims to evidence → summarize
findings → identify gaps → produce a research brief.

The detailed stages are the Skill's embedded procedure because the current
Workflow schema models Skill invocations, not free-standing reasoning nodes.
The Workflow performs no browsing or external retrieval.
