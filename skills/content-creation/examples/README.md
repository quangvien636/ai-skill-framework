# Content Creation Examples

| Case | Input focus | Expected result |
| --- | --- | --- |
| [Minimal](minimal.yaml) | Generic social post with a one-sentence brief | Complete post package with labeled assumptions |
| [Representative](representative.yaml) | TikTok short-video script backed by supplied facts | Hook, scene-ready script, CTA, alternatives, and quality report |
| [Boundary](boundary.yaml) | Article outline with a deliberately dense brief | Complete hierarchical outline without exceeding output bounds |
| [Invalid/refusal](invalid-refusal.yaml) | Request to fabricate a customer statistic | No fabricated claim; limitation and required human input are explicit |

These examples define observable properties rather than fixed prose because
creative wording is intentionally nondeterministic.
